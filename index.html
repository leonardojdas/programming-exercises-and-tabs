Exercise 1

Ask the user for two integers, min and max , and display all the integers from min to max (inclusive of both).

Input

Minimum
Maximum

Exercise 2

Ask the user for a comma-separated set of numeric values. Display the number of values entered, the total, the average, the smallest value, and the largest value.

Input

Numeric Values

Exercise 3

Ask the user for a comma-separated set of numeric values, nums, and for a number to search for, key. Display the number of times key is in nums.

Input

Nums
Key

Exercise 4

Ask the user for a number of stars, numStars, and display that number of stars on a single line. (A star is the '*' character.)

Input

Number of stars

Exercise 5

Ask the user for a number of stars, numStars, and a maximum number of stars per line, max. Display numStars stars with at most max stars per line.

Input

Number of stars
Number per line

Exercise 6

Ask the user for a string. Display the number of vowels (case insensitive) in the string. Display a new string containing a star ('*') in place of each vowel.

Input

String

Exercise 7

At a certain courier company, shipping costs (not including tax) are $5/item for fewer than 100 items, $4/item for between 100 and 1,000 items, and $3/item for more than 1,000 items. Ask the user how many items to ship. Display the base cost, a 15% tax, and the total cost.

Input

Items to ship

Exercise 8

A single Snickers bar costs $1.75, and a box of 24 bars costs $32. Ask the user how many bars they want to buy. Display the number of boxes required, the number of single bars required, the cost to buy the boxes, the cost to buy the single bars, and the total cost.

Input

Bars to buy

Exercise 9

Ask the user to enter a binary number. Convert the binary number to decimal (base 10) and display each step of the conversion as well as the final result. (The steps can be either left-to-right or right-to-left.)

Input

Binary number

Exercise 10

Input

Angle A
Angle B
Angle C
